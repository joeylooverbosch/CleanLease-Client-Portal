## About The Project

This project focuses on designing a client portal for <b>CleanLease</b>, a leading provider of laundry and textile services. The portal aims to enhance the client experience by addressing key pain points across CleanLease's client segments: healthcare (Cure), elderly and mental health facilities (Care), and leisure (Leisure). Using the double-diamond research method, the project has structured research, ideation, and design validation. Deliverables include wireframes and an interactive prototype that reflect client needs and support CleanLease in improving its digital services.
